Complete HTML Laboratory

LAB GOAL
By completing this single laboratory, a learner will be able to:
 Understand HTML from zero level
 Build real-world structured web pages
 Create all types of forms and tables
 Use semantic, accessible, and SEO-friendly HTML
 Be fully ready to move to CSS and JavaScript
